Educational Software Development

In my spare time, I am a software developer. I am eager to apply this skill set to educational and biological topics. For example, I have written software to simulate a population of ficticious animals (snarl-toothed draggles) in Hardy-Weinberg equilibrium, and have designed a module to teach students about the effects that population fragmentation have on genetic drift. This software is freely available to users (see below), and was published in a peer-reviewed journal in November, 2019 (see below). Other software projects can be found on my Github page.
